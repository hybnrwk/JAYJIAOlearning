# SQL 查询分析

## 一句话解释

SQL 查询分析把业务问题翻译成表、字段、粒度、筛选、聚合、关联和窗口函数，是数据分析岗位最基础的数据取数与指标计算能力。

## 前置知识

- 数据库系统架构
- 关系模型
- 数据完整性

## 常见题型

- 按业务口径写聚合、分组、留存、转化和 TopN 查询。
- 判断 JOIN、去重、NULL 和时间范围是否影响指标口径。
- 解释窗口函数、子查询和查询执行顺序。
