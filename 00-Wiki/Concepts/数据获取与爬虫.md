# 数据获取与爬虫

## 一句话解释

数据获取与爬虫关注如何从网页、接口或文件中取得数据，并把非结构化来源整理成可分析的数据。

## 前置知识

- HTTP 与 URL
- HTML 基础
- 文件与数据格式
