# JAYJIAOlearning

这是一个以 Obsidian 笔记和课程资料为主的私人学习仓库，内容覆盖计算机科学、数据科学、数学统计、经管课程与商业分析等方向。

本仓库主要用于个人学习资料的归档、检索与同步，不是一个可运行的软件项目，也不提供统一的构建、部署或使用接口。仓库内容以 Markdown 笔记、PDF 资料、图片附件和少量 Office 文件为主。

## 先叠甲

- 这是私人学习笔记仓库，不是公开课程项目、教学项目、官方资料站或人生指南。
- 笔记来自个人理解、课堂记录、阶段性整理、考前急救和一些精神状态不明的夜晚，可能有错、有漏、有重复、有过时内容。
- 有些笔记确实“又臭又长”：它们可能逻辑绕远、结构松散、标题很认真、正文很意识流。能看懂是缘分，看不懂也不一定是你的问题。
- 仓库里的课件、教材摘录、讲义、图片等主要用于个人学习和备份；相关版权归原作者、课程团队、出版社或其他权利方所有。
- 这里的内容不构成学术、职业、投资、法律或任何正式建议。真要交作业、写论文、做项目，请自行核对原始资料和更可靠来源。
- 目录结构会随着学习进度和整理欲望波动。部分标题可能非常抽象
- 如果发现不适合公开、引用不当或其他问题，请善意提醒，后续会视情况修正、删除或重新整理。

## 文件类型

仓库中主要文件类型包括：

- Markdown：个人笔记、课程总结、知识点整理、复习材料。
- PDF：教材、课件、讲义、课程书籍和部分笔记导出文件。
- 图片：Obsidian 粘贴图片、课程截图、可视化结果图。
- Office 文件：少量 DOCX、PPTX，通常为课程原始资料或作业相关文件。

推荐直接用 Obsidian 打开仓库根目录，以保持 Markdown 笔记、图片附件和相对路径引用的一致性。也可以在 GitHub 网页端按课程目录浏览 Markdown、PDF 和图片文件。

## 使用方式

查找资料时可优先按一级目录定位学科方向，再进入对应课程目录：

- 计算机与数据科学：`cs_ds/`
- 数学与统计：`数统/`
- 经管课程：`经管/`
- 金融科技与商业分析：`金科_ba/`
- 图片附件：`images/`

对于具体知识点，建议使用 Obsidian、编辑器或 GitHub 的全文搜索功能。

## 大文件与 Git LFS

部分大型 PDF 使用 Git LFS 管理。克隆仓库后如需完整获取这些文件，请确保本机已安装 Git LFS，并执行：

```bash
git lfs install
git lfs pull
```

新增文件时建议注意：

- 超过 100 MB 的文件不要直接用普通 Git 提交。
- 大型 PDF 应使用 Git LFS 管理。
- 视频、压缩包、临时文件和大体积中间产物通常不应提交到仓库。
- `.gitignore` 已忽略一批常见无关文件，但新增大文件前仍建议手动检查文件大小和用途。
