## 文件类型

仓库中主要文件类型包括：

- Markdown 笔记：课程笔记、章节总结、自学路线、知识点整理。
- PDF：教材、课件、讲义、课程书籍和部分笔记导出文件。
- 图片：Obsidian 粘贴图片、课程截图、可视化结果图。
- Office 文件：少量 DOCX、PPTX 课程资料。

部分大型 PDF 使用 Git LFS 管理。克隆仓库后如需完整获取这些文件，请确保本机已安装 Git LFS，并执行：

```bash
git lfs install
git lfs pull
```

## 使用方式

推荐直接用 Obsidian 打开仓库根目录，这样可以保持 Markdown 笔记、图片附件和相对路径引用的一致性。也可以在 GitHub 网页端按课程目录浏览 Markdown、PDF 和图片文件。

查找资料时可优先按一级目录定位学科方向，再进入课程目录：

- 计算机与数据科学：`cs_ds/`
- 数学与统计：`数统/`
- 经管课程：`经管/`
- 金融科技与商业分析：`金科_ba/`

## 版本管理说明

- 普通笔记和课程资料直接由 Git 跟踪。
- 超过 GitHub 普通文件限制的大型 PDF 通过 Git LFS 跟踪。
- `.gitignore` 已忽略常见系统文件、临时文件、压缩包、部分数据/代码中间产物和 `*.mp4` 视频文件。
- 新增大文件前建议先检查文件大小；超过 100 MB 的文件不要直接作为普通 Git blob 提交，应改用 Git LFS 或保持本地忽略。

## 注意事项

本仓库是私人学习资料整理仓库，内容会随着课程学习持续调整。部分目录同时包含原始课件、个人笔记和导出资料，命名可能保留课程原始文件名，便于和课堂资料对应。
