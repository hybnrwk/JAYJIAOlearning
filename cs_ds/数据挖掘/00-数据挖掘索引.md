# 数据挖掘索引

数据挖掘目录主要保存周课件、notebook、示例数据和 Jupyter 安装说明。当前以课程资料为主，适合按周次复习数据挖掘流程。

## 学习主线

- 数据理解：课程导论、数据预处理、EDA 与可视化。
- 监督学习：回归、基础分类器、决策树、集成学习、SVM。
- 无监督与规则：关联规则、聚类分析。
- 文本与推荐：文本挖掘、推荐系统、课程总结。

## 核心笔记

- 当前目录暂无 Markdown 主笔记，建议以周课件和 notebook 为主入口。
