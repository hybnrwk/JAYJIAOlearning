# 数据结构与算法索引

数据结构与算法部分分为基础数据结构、常见算法套路、算法设计课程笔记和复习资料。复习时建议先建立结构直觉，再整理算法范式。

## 学习主线

- 基础结构：数组与链表 -> 栈与队列 -> 哈希表 -> 二叉树 -> 图结构。
- 基础算法：排序、字符串/数组处理、树算法、图遍历。
- 算法设计：复杂度分析 -> 图算法 -> 贪心算法 / 分治算法 -> [[00-Wiki/Concepts/动态规划|动态规划]] -> [[00-Wiki/Concepts/网络流|网络流]] -> [[00-Wiki/Concepts/NP完全性|NP 完全性]]。

## 复习抓手

- 数据结构看“操作成本”和“适合维护的信息”。
- 算法范式看“局部选择、递归分解、状态定义、图结构建模和容量约束建模”。
- 期末复习可按“算法续命丹 1-4 -> 算法终极复习 -> 模拟卷”的顺序回收题型。
- 做题时先识别结构，再套算法，不要反过来硬背模板。
