# C++ 程序设计索引

## 学习主线

- 程序设计基础：语法、函数、类、文件和基本数据结构实现。
- 核心编程：对象、内存、容器、字符串和常见编程题。
- 高级程序设计：抽象、递归、搜索、集合、映射和综合题。
