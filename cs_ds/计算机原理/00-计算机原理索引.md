# 计算机原理索引

计算机原理部分当前以张勇《计算机组成原理》作业参考答案和静态存储器实验资料为主。复习时按“数据表示与运算 -> 存储系统 -> 指令与处理器 -> 输入输出”组织知识。

## 学习主线

- 组成基础：数据表示、运算器与指令系统。
- 存储系统：[[00-Wiki/Concepts/存储器层次|存储器层次]] -> Cache、主存与外存。
- 处理器：数据通路、控制器与流水线。
- 实验：[[cs_ds/计算机原理/张勇计组/计算机组成原理课程实验1（静态存储器实验）.pdf|静态存储器实验]]。
