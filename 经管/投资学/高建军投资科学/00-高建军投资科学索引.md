# 高建军投资科学索引

本课程资料覆盖利率与现金流、固定收益、股票估值、投资组合、CAPM、因子模型、效用与风险、远期和期货、期权及其定价。

## 学习主线

- 时间价值：利率与现金流 -> 固定收益与债券定价 -> [[00-Wiki/Concepts/久期|久期]]。
- 权益资产：股票估值 -> [[00-Wiki/Concepts/投资组合与CAPM|投资组合与 CAPM]] -> 因子模型。
- 风险偏好：效用函数与风险度量。
- 衍生品：远期与期货 -> [[00-Wiki/Concepts/期权定价|期权定价]]。

## 复习抓手

- 固收题统一用现金流时间线、贴现因子和利率口径。
- 组合题分清期望收益、方差、协方差和有效前沿。
- 衍生品题先画到期收益，再选择无套利、二叉树或 Black-Scholes 框架。
